## The Vision
**BREW.** is a high-end e-commerce showcase designed with a "Linear" aesthetic. It moves beyond traditional storefronts by integrating a **Gemini-powered AI Sommelier**, creating a sophisticated, personalized shopping experience for coffee enthusiasts.



## Key Features
* **AI Coffee Sommelier:** Real-time recommendations using Google Gemini 3 Flash. It analyzes user flavor profiles to suggest the perfect roast using professional tasting vocabulary.
* **Modern Architecture:** Built on the cutting edge with **React 19** and **TypeScript**, ensuring high-performance state management and type safety.
* **Precision UI/UX:** Features a dark-mode-first design, glassmorphism, background glows, and optimized micro-interactions.
* **Curated Inventory:** Dynamic filtering logic for Roast Levels (Light, Medium, Dark) and detailed "Beyond the Bean" origin storytelling.
* **Secure Checkout Simulation:** A polished, Stripe-inspired payment flow with realistic processing states and validation.

## Technical Stack
* **Framework:** React 19 (Concurrent Mode)
* **Styling:** Tailwind CSS (Custom Animations & Glassmorphism)
* **AI Logic:** Google Generative AI
* **Icons:** Lucide React
* **Bundle Tool:** Vite

## Local Development

### Prerequisites
* **Node.js** (v18.0 or higher)
* **Gemini API Key**

### Installation

1. **Clone the repository and install dependencies:**
   ```bash
   npm install
   ```
2. **Configure Environment Variables**: Create a .env.local file in the root directory and add your Gemini API Key:
   ```bash
   API_KEY=your_gemini_api_key_here
   ```
3. **Start the Development Server:**
   ```bash
   npm run dev
   ```
